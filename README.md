## 🌟 Overview

**MindSync** is a cross-platform mobile application built with **Flutter** that provides a holistic mental wellness experience. Unlike generic wellness apps, MindSync delivers **mood-adaptive personalized screens** — the entire UI and feature set dynamically adjusts based on the user's current emotional state.

The app combines the power of **Google's Gemini AI** for intelligent content generation, **Firebase** for secure real-time data management, and **OpenStreetMap** for live therapist discovery — all wrapped in a calming, hand-crafted "Serenity" design system.

### 🎯 Problem Statement

Mental health apps often provide static, one-size-fits-all content. MindSync solves this by:

- **Personalizing** the experience based on real-time mood input
- **Generating AI-driven** journal prompts, CBT reframing, and wellness goals
- **Tracking progress** with streaks, milestones, and visual analytics
- **Connecting users** to nearby real-world mental health support

### Data Flow Example: Mood Save

```
User taps "Happy" → MoodProvider.saveMood('Happy')
     │
     ├── 1. _selectedMood = 'Happy'
     ├── 2. notifyListeners()          → UI updates immediately
     ├── 3. Firestore.add({mood, timestamp})  → Cloud persistence
     └── 4. fetchUserStats()           → Refresh all statistics
              │
              ├── Count moods (happy/stressed/anxious)
              ├── Count journals, activities
              ├── Calculate streak (consecutive days)
              ├── Check & unlock milestones
              └── notifyListeners()    → Dashboard auto-rebuilds
```

## 🗄 Firebase Database Schema

```
Cloud Firestore
└── users (collection)
    └── {userId} (document)
        ├── moods/           → { mood: "Happy", timestamp: ServerTimestamp }
        ├── journals/        → { prompt: "...", answer: "...", timestamp: ... }
        ├── activities/      → { type: "walking", duration: 600, timestamp: ... }
        ├── goals/           → { title: "...", isCompleted: false, createdAt: ... }
        ├── milestones/      → { title: "3-Day Streak", unlockedAt: ... }
        └── settings/        → { journalEnabled: true, journalHour: 20, ... }
```

**Data Isolation:** Each user's data is stored under their unique Firebase UID (`user.uid`), ensuring complete data privacy and security between users.

| Method | Feature | File |
|--------|---------|------|

### 4. Geolocator (Device)
| Method | Feature | File |
|--------|---------|------|

## 🧮 Custom Algorithms

### Streak Calculation
```
Algorithm: Consecutive-Day Mood Logging
1. Fetch all mood entries sorted by timestamp (descending)
2. For each entry, extract date (ignore time)
3. If difference from previous = 1 day → streak++
4. If difference = 0 → same day, skip
5. If difference > 1 → streak broken
6. Final check: if last entry > 1 day old → streak = 0
```

### Milestone Engine
```
Rule-based achievement system:
• "First Goal Completed"  → Any goal marked complete
• "3-Day Streak"          → streakCount >= 3
• "5 Journal Entries"     → totalJournals >= 5
• "5 Meditation Sessions" → meditationSessions >= 5

Each rule checked on every stats refresh.
New milestones stored in Firestore + popup shown to user.
```

## ⚙️ Setup & Installation

### Prerequisites
- Flutter SDK 3.x
- Dart SDK 3.x
- Firebase CLI
- Android Studio / VS Code
- A Firebase project with Auth + Firestore enabled

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/mindsync.git
cd mindsync

# 2. Install dependencies
flutter pub get

# 3. Configure Firebase
# Ensure lib/firebase_options.dart is valid for your project

# 4. Run the app
flutter run

# 5. (Optional) With Gemini AI key
flutter run --dart-define=GEMINI_API_KEY=your_api_key
```
